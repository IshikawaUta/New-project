Installasi ohmyzsh

OhMyZsh adalah kerangka kerja sumber terbuka untuk mengelola konfigurasi Zsh, Anda bisa menggunakan ohmyzsh untuk menyesuaikan terminal sesuai keinginan dan menambahkan plugin dan fungsi pembantu yang berguna untuk meningkatkan produktivitas dan membuat penggunaan terminal lebih aesthetic.

Command installasi

langkah pertama update terminal

sudo apt update

langkah kedua install git

sudo apt install git

langkah ketiga install zsh

sudo apt install zsh

langkah keempat install ohmyzsh

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/
ohmyzsh/master/tools/install.sh)"

langkah kelima mengatur shell zsh menjadi default

chsh -s $(which zsh)

langkah keenam install termux-ohmyzsh

git clone https://github.com/termux/termux-ohmyzsh.git ~/.oh-my-zsh/custom/plugins/termux

langkah ketujuh memuat ulang konfigurasi zsh

source ~/.zshrc

lalu restart terminal
