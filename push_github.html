Push repository github

Github adalah website yang menyediakan layanan untuk mengembangkan proyek perangkat lunak yang menggunakan sistem kendali versi git, website ini banyak digunakan oleh para programer, karena website ini memiliki banyak fitur salah satunya adalah kolaborasi antar user, pelacakan bug, permintaan fitur, menajemen tugas, wiki proyek, dan bahkan AI copilot.


Command push repository

git init

command ini digunakan untuk membuat repository Git baru atau menginisialisasi ulang repository Git yang sudah ada.

git config --global user.name "masukan username"

command ini digunakan pertama kali untuk mengatur nama pengguna global di Git.

git config --global user.email "masukan email"

command ini digunakan pertama kali untuk mengatur email pengguna global di Git

git add .

command ini digunakan untuk menambahkan semua berkas kedalam repository atau hanya untuk memasukan satu file yang ingin ditambahkan menggunakan command ini.

git add (nama file yang ingin ditambahkan)

git commit -m "masukan komentar"

command ini digunakan untuk menyimpan perubahan pada berkas ke repository lokal.

git branch -M main

command ini digunakan untuk membuat dan mencantumkan cabang.

git remote add origin https://github.com/username/
project-name

(ganti username dan project-name) command ini digunakan untuk menambahkan repository jarak jauh ke repository Git lokal.

git push -u origin main

command ini digunakan untuk mendorong perubahan lokal ke repository daring.
