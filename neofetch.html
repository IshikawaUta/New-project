Installasi neofetch

Neofetch adalah alat informasi sistem yang sudah tidak dikembangkan lagi yang ditulis dalam bahasa skrip shell Bash, tampilan default neofetch, di sisi kiri terdapat logo distribusi, yang ditampilkan dalam seni ASCII, alat ini menampilkan tampilan statis konfigurasi perangkat keras dan perangkat lunak dasar komputer dan versinya, biasanya sistem operasi, host, waktu aktif, manajer paket, shell, resolusi tampilan, lingkungan desktop, manajer jendela, tema dan ikon, terminal komputer, CPU, GPU, dan RAM. Neofetch juga dapat menampilkan gambar di terminal dengan w3m -img atau Sixel sebagai pengganti seni logo ASCII.


Command installasi

langkah pertama update terminal

sudo apt update

langkah kedua install neofetch

sudo apt install neofetch

langkah ketiga jalankan neofetch

neofetch


Konfigurasi ke shell bash

jika anda menggunakan shell bash

buka file .bashrc

nano .bashrc

tambahkan kode ini di bagian akhir

clear
neofetch

lalu tekan ctrl+x pilih y lalu enter

lalu restart terminal


Konfigurasi ke shell zsh

jika anda menggunakan shell zsh

buka file .zshrc

nano .zshrc

tambahkan kode ini di bagian akhir

clear
neofetch

lalu tekan ctrl+x pilih y lalu enter

lalu restart terminal


Mengubah tampilan config

buka file config.conf

nano .config/neofetch/config.conf

hapus semua isi yang ada di file tersebut, lalu copy file ini paste-kan kedalam file tersebut

config.conf

lalu tekan ctrl+x pilih y lalu enter

lalu restart terminal
